Tags HTML não vistas em aula

audio

<audio></audio>

É usada para incorporar conteúdo sonoro em um documento. Essa tag pode conter diferentes fontes de áudio, o navegador escolherá a primeira fonte que ele suportar. O texto entre as tags só será exibido em navegadores que não suportam o elemento.

blockquote

<blockquote></blockquote>

Usada para realizar citações de conteúdos disponíveis em outros sites ou ainda outras fontes, como por exemplo: trechos de livros ou textos impressos. Tem como atributo a tag <cite></cite> que especifica a origem da citação.

code

<code></code>

É usada para definir um código de computador. O conteúdo é exibido na fonte monospace padrão do navegador.

Ver mais

description list

<dl></dl>

Define uma lista de descrição. Normalmente usada em sites de significados de palavras. É usada em conjunto com: <dt></dt> que define o termo/nome; e <dd></dd> que descreve cada termo/nome.

iframe

<iframe></iframe>

É responsável por incorporar no site o conteúdo de uma página externa. Ao invés de tirar o usuário da página (utilizando <a></a>) para visualizar determinado conteúdo, podemos trazer o conteúdo para o nosso site.

Ver mais

table

<table></table>

Define uma tabela no HTML. Sua formação básica é composta por linhas <tr></tr>, colunas <td></td> e cabeçalhos <th></th>.

Anatomia básica de um documento HTML
NOME DESCRIÇÃO
<!DOCTYPE html> É a declaração que define que este é um documento HTML5.
<html> É o elemento raiz de uma página HTML, contêm todos os outros elementos filhos.
<head> Contêm informações adicionais sobre a página. Recebe os links para folha de estilo, palavras-chaves para indexação, descrição da página e título.
<title> Responsável por nomear a aba do navegador.
<body> Define o corpo do documento e é o container para todo o conteúdo visível, como: títulos, parágrafos, imagens, hyperlinks, tabelas, listas, etc.
